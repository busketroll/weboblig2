Git og GitHub

Om Git

Git, laget av Linus Torvalds er et programvare fyllt med repositorer som ble designet for å være hurtig og sammenflettende. Git er alle-til-alle-synkroniserende, slik at alle som er innvolvert kan gjøre endringer. Gjennom kommandoene "pull" og "push" kan endringer i programmet bli sett av alle før det eventuelt, ved trykk av en enkel knapp "pushet" til publisitet. Hovedpoenget med git er å støtte desentralisert arbeidsflyt.

Om GitHub

GitHub er en web-basert Git som er hovedsakelig ment for kode. GitHub inneholder alle de funksjonelle delene som Git tilbyr, samtidig som den legger til sine egene trekk. Repositorene dine i GitHub kan enten være private eller åpne for mange. GitHub har over 20 millioner brukere og over 57 millioner repositorer.
