MathML

Om MathML

MathML er en form for spesifikasjon for mattematiske formler, begreper og utrykk på web. Det står for Mathematical Markup Language og kan strukturere mattematiske notasjoner på en mate ingen andre programmer kan. MathML er laget av W3C og ble først publisert i 1998 som det første XML språket som W3C anbefalte. Ved bruk av MathML kan semantiske formler bli presentert på web gjennom svært enkel HTML. Koder som for identifikasjon, for operatorer og for nummer er kun noen få av mange koder som MathML bruker.
