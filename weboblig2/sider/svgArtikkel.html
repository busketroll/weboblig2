SVG

Om SVG

SVG (Scalable Vector Graphics) er et format for tegning av to-dimensjonal grafikk. SVG er støttet for interaktivitet og animasjon. Siden SVG er XML-basert så er oppførselen deres definert av XML-formater, det betyr at de kan bli scripted og komprimert. SVG kan bli laget med all slags tekst-editor program og andre tegneprogrammer. Alle moderne browsere støtter SVG, til og med Internett Explorer.

SVG bruker disse 14 funksjonelle områder for å gjenomføre arbeidet: Paths, Basic Shapes, Text, Painting, Color, Gradients og Patterns, Clipping (Masking and Compositing), Interactivity, Linking, Scripting, Animation, Fonts og Metadata. Med SVG kan man tegne figurer, former og masse annet i kodeprogrammer, slik som notepad++. Man kan også lage animasjoner ved hjelp av CSS gjennom @keyframes. Denne metoden for animasjon er veldig simpel og einer seg kun for de enkleste animasjoner da SVG tegningen kan ha vansker med å gjøre kompliserte bevegelser uten at koden blir svært lang.
